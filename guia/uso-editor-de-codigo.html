Instalación y uso de un editor de codigo

Un buen editor de codigo nos ayudara a simplificar y mejorar su codigo legible, nos dira donde estan los errores, validar y en algunos ejecutar su codigo por usted.

En esta guía para los ejemplos y capitulos vamos a utilizar el editor Visual Studio Code, es un editor de codigo fuente muy ligero y potente que soportar diferentes lenguajes de programación como: (.JS, C ++, C #, Python, PHP, Go) y runtimes (como .NET y Unity), adaptable para Windows, macOS y Linux. Muy recomendado.



Como instalar Visual Studio Code

Seguir estos pasos:

  • Ingresar a la pagina web Visual Studio Code.
  • Descargar la version que se adapta a su PC:
  • Una vez descargado, ejecute el instalador (VSCodeSetup-version).exe, esto sólo tomará unos minuto.
Pasos adicionales que le ayudarán a aprender más sobre VS Code:
(Estos pasos solo son recomendaciones para mejora, no es necesario seguirlos.)

  • Interfaz de usuario - Una orientación rápida al código VS.
  • Componentes adicionales : Aprenda a instalar Git, TypeScript y herramientas como Yeoman.
  • Guia VS code Edicion basica.
  • Mas detalle sobre el uso y interfaz de VS Code VSCode-interface.


Uso de Visual Studio Code

VS Code comienza con una interfaz sencilla con una agradable pantalla de bienvenida y una guía de uso. Tiene la capacidad de abrir una carpeta de proyecto para que no tenga que seguir abriendo los archivos individualmente.

  • Abrir VS Code.
  • Vaya a Archivo, Abrir carpeta (CTRL + K).



  • Ubicar la carpeta principal y haga clic Seleccionar Carpeta (Sus archivos no aparecerán, sólo la estructura de la carpeta)
A la izquierda del editor tendrás todos los archivos. Simplemente haga doble clic en cualquiera de ellos para visualizar y empezar a editar. Para limpiar las cosas, puede cerrar la pantalla de bienvenida y tal.
