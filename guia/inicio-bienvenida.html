Bienvenido

Bienvenido a la guia MyBOT, en esta guia utilizaremos Discord.js v11.3.2 que es la version que ofrece una mejor estabilidad, así como soporte para nuevas funciones como registros de auditoría y búsqueda de mensajes.

  • ¿Que es Discord.js?

    • Es un potente módulo Node.js que te permite interactuar con Discord API muy fácilmente.

  • ¿Que es Node.js?

    • Es un entorno de ejecución para JavaScript, usa un modelo de operaciones E/S sin bloqueo y orientado a eventos, que lo hace liviano y eficiente.

  • ¿Que es Discord API?

    • Es una interfaz de desarrollo para potenciar sus aplicaciones y crear bots para sus comunidades.
